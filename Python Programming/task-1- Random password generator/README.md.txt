# Random Password Generator

## Project Overview

This project was completed as part of the Oasis Infobyte Python Programming Internship.

The objective of this project is to create a Python program that generates random passwords based on user-defined choices such as password length, letters, numbers, and symbols.

The project also includes a password strength checker and a cyber security tip to encourage safer password habits.

## Cyber Security Relevance

Weak passwords are a common security risk because they can make accounts easier to guess, crack, or brute-force. This project shows how Python can be used to generate stronger passwords by combining different character types and encouraging users to create longer and more unique passwords.

## Features

* Allows the user to enter a password length
* Allows the user to include letters
* Allows the user to include numbers
* Allows the user to include symbols
* Generates a random password
* Checks the password strength
* Provides a password safety tip
* Handles invalid user input

## Technologies Used

* Python
* random module
* string module

## How to Run the Project

1. Open the `Random Password Generator` folder.
2. Open a terminal inside the folder.
3. Run the command below:

```bash
python password_generator.py
```

If that does not work, use:

```bash
py password_generator.py
```
